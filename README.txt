Speed Data
==========

	An android application for data collection of vehicle traffic

Requirement
-----------
 Record the times a vehicle crosses two lines and to estimate the speed

Funcitonality
------------

   The application has two main activities : Start sevice and stop service 

   Once the service is started it creates a file named by the current time to record data with a directory named by current date.
   Once a vehicle icon is clicked then the time it crossed the first line is recorded and rest icons become unclickable.
   Once the above vehicle crosses the second line and again its icon is clicked and time is recorded and whole data written into a file.
   After this, all icons become clickable again for further data collection
